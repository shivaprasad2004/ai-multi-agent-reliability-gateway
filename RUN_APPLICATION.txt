TROUBLESHOOTING:
───────────────
Connection Refused?
  1. Run: FIX_CONNECTION.bat
  2. Or: docker-compose down && docker-compose up --build -d
  3. Wait 60 seconds, then check: docker-compose ps
  4. Check logs: docker-compose logs frontend

Port in use?      docker-compose down
Services restarting? Check logs: docker-compose logs
Can't connect?    Wait longer, check docker-compose ps

See IMMEDIATE_FIX.md for detailed troubleshooting
